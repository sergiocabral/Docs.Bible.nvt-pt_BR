[[v26-15-0]]
=== Ezequiel, 15
// nvt ?, NVT - Nova Versão Transformadora

[[v26-15-1]]

^<<v26-15-1,1>>^ Recebi esta mensagem do SENHOR:

[[v26-15-2]]

^<<v26-15-2,2>>^ "Filho do homem, como comparar uma videira com uma árvore? Acaso a madeira da videira é útil como a da árvore?

[[v26-15-3]]

^<<v26-15-3,3>>^ Pode ser usada para fazer objetos, como ganchos para pendurar vasilhas e panelas?

[[v26-15-4]]

^<<v26-15-4,4>>^ Não, ela serve apenas como lenha e, mesmo assim, queima depressa demais.

[[v26-15-5]]

^<<v26-15-5,5>>^ As videiras são inúteis tanto antes como depois de serem lançadas no fogo.

[[v26-15-6]]

^<<v26-15-6,6>>^ "Assim diz o SENHOR Soberano: Os habitantes de Jerusalém são como videiras que crescem no meio das árvores do bosque. Porque são inúteis, eu os lancei no fogo para serem queimados.

[[v26-15-7]]

^<<v26-15-7,7>>^ E, se escaparem de um fogo, providenciarei outro para os consumir. Quando eu me voltar contra eles, vocês saberão que eu sou o SENHOR.

[[v26-15-8]]

^<<v26-15-8,8>>^ Deixarei a terra desolada, pois meu povo foi infiel a mim. Eu, o SENHOR Soberano falei!".

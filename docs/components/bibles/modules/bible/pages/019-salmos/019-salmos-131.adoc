[[v19-131-0]]
=== Salmos, 131
// nvt ?, NVT - Nova Versão Transformadora

[[v19-131-1]]

^<<v19-131-1,1>>^ SENHOR, meu coração não é orgulhoso, e meus olhos não são arrogantes. Não me envolvo com questões grandiosas ou maravilhosas demais para minha compreensão.

[[v19-131-2]]

^<<v19-131-2,2>>^ Pelo contrário, acalmei e aquietei a alma, como criança desmamada que não chora mais pelo leite da mãe. Sim, minha alma dentro de mim é como uma criança desmamada.

[[v19-131-3]]

^<<v19-131-3,3>>^ Ó Israel, ponha sua esperança no SENHOR, agora e para sempre!

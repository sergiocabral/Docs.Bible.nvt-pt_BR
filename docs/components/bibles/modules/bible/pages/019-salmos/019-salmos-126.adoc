[[v19-126-0]]
=== Salmos, 126
// nvt ?, NVT - Nova Versão Transformadora

[[v19-126-1]]

^<<v19-126-1,1>>^ Quando o SENHOR trouxe os exilados de volta a Sião, foi como um sonho.

[[v19-126-2]]

^<<v19-126-2,2>>^ Nossa boca se encheu de riso, e cantamos de alegria. As outras nações disseram: "O SENHOR fez coisas grandiosas por eles".

[[v19-126-3]]

^<<v19-126-3,3>>^ Sim, o SENHOR fez coisas grandiosas por nós; que alegria!

[[v19-126-4]]

^<<v19-126-4,4>>^ Restaura, SENHOR, nossa situação, como os riachos revigoram o deserto.

[[v19-126-5]]

^<<v19-126-5,5>>^ Os que semeiam com lágrimas colherão com gritos de alegria.

[[v19-126-6]]

^<<v19-126-6,6>>^ Choram enquanto lançam as sementes, mas cantam quando voltam com a colheita.

[[v19-125-0]]
=== Salmos, 125
// nvt ?, NVT - Nova Versão Transformadora

[[v19-125-1]]

^<<v19-125-1,1>>^ Os que confiam no SENHOR são como o monte Sião; não serão abalados, mas permanecerão para sempre.

[[v19-125-2]]

^<<v19-125-2,2>>^ Assim como os montes cercam Jerusalém, o SENHOR se põe ao redor de seu povo, agora e para sempre.

[[v19-125-3]]

^<<v19-125-3,3>>^ Os perversos não governarão a terra dos justos, para que os justos não sejam tentados a fazer o mal.

[[v19-125-4]]

^<<v19-125-4,4>>^ Ó SENHOR, faze o bem àqueles que são bons, aos que têm o coração sincero.

[[v19-125-5]]

^<<v19-125-5,5>>^ Expulsa, porém, os que andam por caminhos tortuosos, ó SENHOR; leva-os embora junto com os que praticam o mal. Que Israel tenha paz!
